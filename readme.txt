Нужно:
nodejs
npm,
mongodb (как установить и запускать - https://metanit.com/nosql/mongodb/1.2.php)

Как запустить -
два консольных окна, + запуск mongod.exe в папке bin самого mongodb
в первом консольном окне заходим в папку с проектом и пишем npm install и установки пакетов
после того как пакеты установятся, пишем npm run start

во втором консольном окне заходим my-app/backend пишем node init.js (инициализация бд), когда
выполнится пишем nodemon index.js
и того три консольных окна, на которых крутится сервер приложения, сервер бд, и сам сервер
пользуемся
